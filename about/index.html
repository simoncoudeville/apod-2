About

This is a personal project to practice my design, CSS & javascript skills. I'm not necessarally a space fan but it seemed like a interesting subject to work with.

All data is fetched from the Astronomy Picture of the Day Api and displayed using vanilla javascript. The rest is build with good old HTML & CSS. Explore the code on Github.

The text is set in Xanh Mono regular & italic. A monospaced serif typeface, designed by Lam Bao and Duy Dao.
